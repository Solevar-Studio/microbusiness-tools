# microbusiness-tools

Лінійка легких вебінструментів для мікробізнесів. Кожен інструмент — окремий статичний застосунок без бекенду, згрупований за доменом і типом бізнесу.

## Структура

```
microbusiness-tools/
└── horeca/            — готелі, ресторани, кавʼярні, пекарні, кейтеринг
    ├── bakery/        — пекарня: додаток-компаньйон (медовик), order-flow
    └── sushi/         — заплановано
```

## Конвенції

- lowercase, kebab-case; папка верхнього рівня — домен (`horeca`), нижче — тип бізнесу (`bakery`, `sushi`).
- кожен інструмент містить `README.md`, робочий `index.html` і, за наявності, `CASE-STUDY.md` та `PROMPT.md`.
- персональні дані клієнтів у код не закладаються; дія веде в приватний канал виробника.
- живі демо підпапок — через Netlify / Cloudflare Pages із base-dir на папку інструмента (GitHub Pages віддає лише корінь або `/docs`).
