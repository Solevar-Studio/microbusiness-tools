# Пекарня — додаток-компаньйон (order-flow)

Легкий односторінковий застосунок для пекарні-виробника. Дає покупцю рецепт, покрокову збірку та живий розрахунок ще до звернення і веде замовлення в один крок у месенджер виробника. Без бекенду, розгортається як статичний сайт.

Перша реалізація — медові коржі для домашнього медовика: тримовний застосунок (UA / EN / NL), розгорнутий для локального виробника. Аналіз, рішення та безпека описані в `CASE-STUDY.md`; специфікація для відтворення — у `PROMPT.md`.

Шлях у репозиторії: `microbusiness-tools/horeca/bakery/`.

## Вміст

```
bakery/
├── README.md      — цей файл
├── CASE-STUDY.md  — кейс: аналіз, рішення, безпека, результат
├── PROMPT.md      — специфікація для відтворення застосунку
├── index.html     — застосунок (заглушки замість контактів)
└── og.jpg         — прев'ю-картинка для соцмереж (додайте власну, див. нижче)
```

## Заглушки перед запуском

У `index.html` замініть:

| Заглушка | На що |
|---|---|
| `YOUR_WHATSAPP` | номер WhatsApp у міжнародному форматі, без «+» (напр. `380XXXXXXXXX`) |
| `YOUR_INSTAGRAM` | інстаграм-хендл виробника-партнера |
| `[КОНДИТЕРКА]` / `[BAKER]` | ім'я виробника (UA / EN, NL) |
| `[СТУДІЯ]` | ваш підпис розробника у футері |
| `YOUR-SITE.example` | домен розгортання (для OG-тегів) |

`og.jpg` в архів не входить. Додайте власну прев'ю-картинку в цю папку і переконайтесь, що тег `og:image` в `index.html` вказує на неї. Без файлу картка при поділі лінка відобразиться без зображення.

## Розгортання

1. Живе демо підпапки моно-репо — через Netlify / Cloudflare Pages із base-dir на `horeca/bakery/`. GitHub Pages напряму цю підпапку не віддасть (лише корінь або `/docs`).
2. Заповніть OG-теги реальним доменом і додайте `og.jpg` для коректної прев'ю-картки.

## Технологія

HTML + CSS + JS без фреймворків, зображення вбудовані (base64), зовнішні залежності — лише вебшрифти. Персональні дані клієнтів не збираються: дія веде в месенджер виробника.

## Ліцензія / атрибуція

Шаблон навчальний. Підпис розробника у футері (`[СТУДІЯ]`) замініть на власний.
